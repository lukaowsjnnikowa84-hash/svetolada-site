Анализ выполняется по направлениям:

  • 🔢 Нумерология
  • 🌞 Астрология
  • 🌿 Ведическая астрология (джйотиш)
  • 💛 Аюрведа
  • 🌀 Психология

Для анализа используются ваши персональные данные:
• Дата рождения — формат дд.мм.гггг
• Время рождения — можно задать точно (чч:мм), выбрать период (утро, день, вечер, ночь), или не задавать вовсе. Если время не указано — анализ всё равно будет выполнен, но некоторые моменты будут исключены.
• Место рождения — можно указать город/страну. Задание этого показателя повышает точность анализа.

Для анализа совместимости с партнёром нужно указать его (её) дату рождения в формате дд.мм.гггг. Если партнёр не указан — анализ будет выполнен, и система покажет портрет идеального для вас партнёра.
