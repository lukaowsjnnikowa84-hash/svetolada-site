Политика конфиденциальности

1. Общие положения

Данная Политика конфиденциальности описывает, какие данные собирает проект «Светолада», как они используются, хранятся и защищаются.

2. Какие данные собираются

Для формирования персонального отчёта вы предоставляете:
— имя
— дату рождения
— пол

3. Как используются данные

Данные применяются только для генерации персонального отчёта и не передаются третьим лицам.

4. Хранение данных

Данные не сохраняются на постоянной основе и удаляются автоматически после формирования отчёта.

5. Защита данных

Передача данных осуществляется по защищённому соединению (HTTPS).

6. Файлы отчётов

Сгенерированные PDF‑файлы хранятся временно и автоматически удаляются через некоторое время.
