# 🌿 Светолада — персональные световые отчёты

**Светолада** — это тёплый, мягкий сервис, созданный для того, чтобы помогать человеку увидеть свою внутреннюю красоту, особенности, природные дары и путь к гармонии.  
Проект объединяет нумерологию, астрологию, ведические знания, психологию и элементы аюрведы.

---

## ✨ Возможности проекта

- 📄 Генерация персонального PDF‑отчёта
- 🔢 Нумерологический анализ
- 🌞 Астрологические характеристики
- 🌿 Ведическая астрология (джйотиш)
- 💛 Аюрведические рекомендации
- 🌀 Психологические описания
- 💞 Анализ совместимости или портрет идеального партнёра

---

## 🧱 Структура проекта

svetolada-site/
│
├── frontend/ # Веб-интерфейс
│ ├── assets/ # HTML-страницы
│ ├── js/ # JavaScript-логика
│ └── styles/ # CSS (если появится)
│
├── generator/ # Генерация PDF
│ ├── resources/ # Картинки для PDF и сайта
│ └── templates/ # Шаблоны PDF
│
├── pdf/ # Готовые PDF (опционально)
│
└── backend/ # Серверная часть (Flask)

---

## 🖼 Подключение ресурсов

HTML‑страницы лежат в `frontend/assets/`,  
картинки — в `generator/resources/`.

Пример правильного пути:

```css
background: url("../../generator/resources/background.png");
JS‑файлы подключаются так:
<script src="../js/result.js"></script>
🚀 Запуск проекта
1. Установка зависимостей backend
pip install -r requirements.txt
2. Запуск сервера
python backend/app.py
Сервер поднимается на:
http://localhost:5000
3. Открытие фронтенда
Открыть файл:
frontend/assets/index.html
📄 Генерация PDF
Фронтенд отправляет запрос:
fetch("http://localhost:5000/generate", {
  method: "POST",
  body: JSON.stringify(data)
})
Backend формирует PDF с использованием шаблонов из:
generator/templates/
```
