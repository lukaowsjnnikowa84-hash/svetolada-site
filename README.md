# Светолада 🌿
Тёплый сервис персональных отчётов, созданных с заботой и вниманием.

## Описание
Светолада — это онлайн‑платформа, где человек может создать персональный PDF‑отчёт.  
Первая версия полностью бесплатна и включает:
- уютный фронтенд (HTML, CSS, JS);
- сервер на Python (Render.com);
- генерацию PDF‑отчёта.
